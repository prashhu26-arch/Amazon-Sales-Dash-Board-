📊 Amazon Sales Dashboard
This project presents an interactive Power BI dashboard analyzing Amazon sales data to generate meaningful business insights.

📈 Key Insights

- Sales trends over time (monthly/yearly)
- Profit analysis by category and region
- Top-performing products and cities
- KPI metrics like total sales, profit, and growth %

🛠 Tools Used

- Power BI
- Excel / CSV dataset

🔗 Project Link

https://github.com/prashhu26-arch/Amazon-Sales-Dash-Board-.git

🚀 Outcome

This project helps in understanding business performance and making data-driven decisions.
